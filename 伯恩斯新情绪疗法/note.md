《伯恩斯新情绪疗法》阅读记录

# 1.情绪障碍症治疗的最新突破

给出了一组对比实验的数据，数据表明**认知疗法**的效果比**药物治疗**更好。  

我很自然地怀疑数据的真实性，然后在想，我应该以什么样的心态来读这本书，要保持批判性思维吗？我应该宽容地获得更多可以安慰到我的东西还是要严肃地获取我无法推翻的观点？这本书要给我的是什么？

接下来基于这份实验数据作者写了一些话，我完全不想看。

# 2.诊断情绪：治疗的第一步

伯恩斯抑郁情况自查表

狂躁症与抑郁症相反，会使人喜怒无常、自信心过度膨胀、极度活跃、睡眠需求减少。有时可能会突然转化为深深的抑郁。

# 3.破解情绪：思维决定情绪

**Q：抑郁症的根源是什么？**  

A：“其中大部分原因在于不合逻辑的悲观态度”

消极思维是自挫情绪的根源，也是一种抑郁症状



感受来自于思维而不是真正发生的事。所有的感觉都必须先由头脑处理赋予主观意义，之后才会产生情绪反应。

**10大认知扭曲**

+ 非此即彼思维
+ 以偏概全
+ 心理过滤
+ 否定正面思考
+ 妄下结论
+ 放大和缩小
+ 情绪化推理
+ “应该”句式
+ 乱贴标签
+ 罪责归己

认知扭曲自评测试

# 4.从建立自尊开始

亚伦·贝克的研究表明80%以上的抑郁患者都有自厌情绪。抑郁患者越是渴求一些优势，例如智慧、成就、声望、魅力、健康和力量，就越认为自己在这些方面极为匮乏。他认为抑郁患者的自我形象可以用4个D来表示：一无所成(Defeated)，一无是处(Defective)，一无所依 (Deserted)，一无所有(Deprived)。

> 题外话，我发现书里用“实验”表明观点时，如果不向我展示实验的具体数据，那我不会对这个实验产生任何怀疑

**提升自尊的特效方法**

+ 反驳自我批评
  + 自我批评念头闪过时，把它们识别记录下来
  + 找出这些思维扭曲的根源
  + 反驳它们，从而培养一种更客观的自我评价体系
+ 心理生物反馈
  + 高尔夫计数器
+ 行动起来，不要自怨自艾
  + 前两个方法可以同时使用

# 5.战胜无为主义
